SiteSearch Demo - Custom Configuration

This demo shows how to customize SiteSearch with advanced options.

About This Demo

This demo showcases various customization options available in SiteSearch:

  • Custom button instead of the default floating button
  • Custom colors and text
  • Custom data and callbacks

Implementation Code

Here's the code used to implement this demo:

<!-- Custom search button in HTML -->
<button id="custom-search-button" class="custom-search-button">Search Documentation</button>

<!-- SiteSearch Integration with Custom Configuration -->
<script>
    window.siteSearchOptions = {
        // Appearance - custom colors
        showSearchButton: false, // Hide the default floating button
        searchButtonSelector: '#custom-search-button', // Use a custom button instead
        accentColor: '#0984e3',
        secondaryColor: '#0652DD',
        
        // Custom text
        searchPlaceholder: 'Search the documentation...',
        popupTitle: 'Documentation Search',
        noResultsText: 'No documentation found for your query',
        loadingText: 'Searching documentation...',
        
        // Custom test data
        dataSource: 'test',
        testData: [
            {
                id: "custom-1",
                title: "Custom Configuration Guide",
                content: "This guide explains how to customize SiteSearch...",
                url: "#custom-config",
                type: "guide",
                date: "2025-03-15"
            },
            // More test data...
        ],
        
        // Custom callback for search
        onSearch: function(term, results) {
            console.log('Search performed:', term);
            console.log('Results found:', results.length);
        },
        
        // Custom callback for result click
        onResultClick: function(result) {
            console.log('Result clicked:', result);
            // You can implement custom navigation or other actions here
            window.location.href = result.url;
        }
    };
</script>
<script src="site-search-integration.js"></script>

Available Customization Options

Appearance

  • showSearchButton: Show/hide the default floating search button
  • searchButtonPosition: Position of the search button ('bottom-right', 'bottom-left', 'top-right', 'top-left')
  • searchButtonColor: Background color of the search button
  • accentColor: Primary accent color for the search interface
  • secondaryColor: Secondary color for hover states and highlights
  • searchButtonSelector: CSS selector for a custom search button

Text Customization

  • searchPlaceholder: Placeholder text for the search input
  • popupTitle: Title of the search popup
  • noResultsText: Text shown when no results are found
  • loadingText: Text shown while loading content
  • errorText: Text shown when an error occurs
  • searchButtonText: Text/icon for the search button

Callbacks

  • onSearch: Called when a search is performed
  • onResultClick: Called when a search result is clicked
