SiteSearch Demo - Local JSON

This demo shows how to use SiteSearch with a local JSON file.

About This Demo

This demo uses a local JSON file as the data source for SiteSearch. This is useful for:

  • Static websites without a backend
  • Small to medium-sized websites with infrequently changing content
  • Testing and development environments

Click the search button in the bottom-right corner to try it out!

Implementation Code

Here's the code used to implement this demo:

<!-- SiteSearch Integration -->
<script>
    window.siteSearchOptions = {
        // Appearance
        showSearchButton: true,
        searchButtonPosition: 'bottom-right',
        searchButtonColor: '#6c5ce7',
        accentColor: '#a29bfe',
        secondaryColor: '#5758BB',
        
        // Data source - using local JSON file
        dataSource: 'json',
        dataEndpoint: 'sample-data.json'
    };
</script>
<script src="site-search-integration.js"></script>

JSON Data Format

The JSON file should contain an array of objects with the following properties:

[
  {
    "id": "1",
    "title": "Article Title",
    "content": "Article content goes here...",
    "url": "/article.html",
    "type": "article",
    "date": "2025-03-01"
  },
  ...
]

You can view the complete sample-data.json file used in this demo.
