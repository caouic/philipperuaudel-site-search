SiteSearch Demo - API Integration

This demo shows how to use SiteSearch with an API.

About This Demo

This demo shows how to connect SiteSearch to an API for dynamic content. This is useful for:

  • Dynamic websites with frequently changing content
  • Large websites with extensive content
  • Content management systems and headless CMS setups
Note: For demonstration purposes, this demo uses a local JSON file that simulates an API response. In a real implementation, you would use an actual API endpoint.

Click the search button in the bottom-right corner to try it out!

Implementation Code

Here's the code used to implement this demo:

<!-- SiteSearch Integration -->
<script>
    window.siteSearchOptions = {
        // Appearance
        showSearchButton: true,
        searchButtonPosition: 'bottom-right',
        searchButtonColor: '#e17055',
        accentColor: '#ff7675',
        secondaryColor: '#d63031',
        
        // API configuration
        dataSource: 'api',
        dataEndpoint: 'https://api.example.com/search-content',
        apiKey: 'your-api-key-here'
    };
</script>
<script src="site-search-integration.js"></script>

API Response Format

SiteSearch supports various API response formats, but the most common is:

{
  "status": "success",
  "data": [
    {
      "id": "1",
      "title": "Article Title",
      "content": "Article content...",
      "url": "/article.html",
      "type": "article",
      "date": "2025-03-01"
    },
    ...
  ]
}

You can view the complete sample-api-response.json file used in this demo.

Using with Different API Providers

SiteSearch is designed to work with various API providers. Here are some examples:

1. Kernex API

window.siteSearchOptions = {
    dataSource: 'api',
    dataEndpoint: 'https://api.kernex.io/api/v1/your-project-id/resource/your-resource/:resourceId',
    apiKey: 'your-kernex-api-key',
    resourceId: 'your-resource-id'
};

2. Custom REST API

window.siteSearchOptions = {
    dataSource: 'api',
    dataEndpoint: 'https://your-domain.com/api/search-content',
    apiKey: 'your-api-key'
};

3. Headless CMS (e.g., Contentful, Strapi)

window.siteSearchOptions = {
    dataSource: 'api',
    dataEndpoint: 'https://cdn.contentful.com/spaces/your-space-id/environments/master/entries',
    apiKey: 'your-contentful-access-token'
};
