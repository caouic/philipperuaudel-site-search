Easy Integration

Add powerful search functionality to any website with just a few lines of code. No server-side setup required!

<!-- Add to your <head> section -->
<script>
    window.siteSearchOptions = {
        dataSource: 'json',
        dataEndpoint: 'your-data.json'
    };
</script>
<script src="site-search-integration.js"></script>

Flexible Data Sources

Connect to various data sources:

  • Built-in test data - Great for prototyping
  • Local JSON files - Perfect for static sites
  • Custom APIs - For dynamic content
  • Kernex API - Easy integration with Kernex
  • Headless CMS - Works with Contentful, Strapi, etc.
